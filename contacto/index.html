¿Tienes dudas sobre un componente, una guía técnica o un pedido?

Nos comprometemos a responder en **menos de 24 horas hábiles**.

Canal adecuado para tu consulta

  • Para ventas: Preguntas sobre disponibilidad, precios, cotizaciones o pedidos. → ventas@primeimportsspa.cl
  • Para soporte técnico: Ayuda con autenticación de componentes, uso de guías, código o diagramas. → soporte@primeimportsspa.cl
  • Para consultas generales: Facturación, envíos, políticas o colaboraciones. → info@primeimportsspa.cl

¿Qué incluir en tu correo?

Para ayudarte más rápido, por favor indícanos:

  • El modelo del producto (ej: ESP32-WROOM-32E, LRS-50-12).
  • El problema específico que estás enfrentando.
  • Si aplica, una foto del componente o el código que estás usando.
