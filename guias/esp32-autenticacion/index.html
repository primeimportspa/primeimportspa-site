Cómo identificar un ESP32-WROOM-32E original

Basado en el datasheet oficial de Espressif (2025)

El módulo ESP32-WROOM-32E es ampliamente falsificado en el mercado chileno. Esta guía te enseña a distinguir un módulo original usando criterios técnicos verificables.

1. Características físicas (según Espressif)

El datasheet oficial (Sección 1.1, Figura 1) describe las siguientes características del módulo auténtico:

  • Chip principal: debe decir claramente “ESP32-D0WD” con el logo de Espressif.
  • Flash SPI: chip de 4 MB de marcas autorizadas: “Winbond W25Q32JV” o “ISSI IS25LP032D”.
  • Regulador de voltaje: “AMS1117-3.3” (marca “AMS” o “1117”).
  • PCB: color verde oscuro, con serigrafía nítida y el código “WROOM-32E” en la parte inferior.

⚠️ Nota: Las imágenes en esta guía son de referencia, extraídas del datasheet oficial de Espressif. Al recibir nuestros módulos, actualizaremos con fotos reales.

2. Prueba técnica con Arduino

Un módulo original reportará correctamente su modelo y flash mediante la API oficial de Espressif. Sube este código:

#include "esp_chip_info.h"
#include "esp_flash.h"

void setup() {
  Serial.begin(115200);
  while (!Serial);

  esp_chip_info_t chip_info;
  esp_chip_info(&chip_info);

  Serial.println("=== Información del chip ESP32 ===");
  Serial.print("Cores: ");
  Serial.println(chip_info.cores);
  Serial.print("Tipo: ");
  if (chip_info.model == CHIP_ESP32) Serial.println("ESP32");
  else Serial.println("Desconocido");

  Serial.print("Revisión: ");
  Serial.println(chip_info.revision);
  Serial.print("Flash: ");
  Serial.print(esp_flash_get_size(NULL) / (1024 * 1024));
  Serial.println(" MB");
}

void loop() {
  delay(10000);
}

Un módulo original mostrará:

=== Información del chip ESP32 ===
Cores: 2
Tipo: ESP32
Revisión: 1
Flash: 4 MB

Las réplicas suelen mostrar “0 MB”, “Desconocido” o reiniciarse al ejecutar este código.

3. ¿Por qué importa usar el original?

  • ✅ Soporte oficial: actualizaciones del SDK Arduino-ESP32 y ESP-IDF.
  • ✅ 4 MB de flash certificada: no 2 MB como en muchas réplicas.
  • ❌ Las réplicas carecen de certificaciones FCC/CE y tienen problemas con OTA, Bluetooth y sensores internos.
