Automatiza tu huerto con ESP32 y DHT22

Publicado el 5 de noviembre de 2025 · Por Prime Imports SpA

En este tutorial, construiremos un sistema de monitoreo ambiental para un huerto casero, usando componentes 100% originales y código probado.

Materiales necesarios

  • ESP32-WROOM-32E original
  • Sensor DHT22 (con resistencia pull-up integrada)
  • Fuente Mean Well LRS-50-12
  • Protoboard y jumpers

Diagrama de conexión

(Aquí irá una imagen cuando tengas el kit armado. Por ahora, descripción textual):

  • ESP32 GPIO4 → Data del DHT22
  • DHT22 VCC → 3.3V del ESP32
  • DHT22 GND → GND del ESP32

Código fuente

#include "DHT.h"
#define DHTPIN 4
#define DHTTYPE DHT22

DHT dht(DHTPIN, DHTTYPE);

void setup() {
  Serial.begin(115200);
  dht.begin();
  Serial.println("Monitoreo de huerto iniciado");
}

void loop() {
  float h = dht.readHumidity();
  float t = dht.readTemperature();

  if (!isnan(h) && !isnan(t)) {
    Serial.print("Humedad: "); Serial.print(h); Serial.print("% ");
    Serial.print("Temperatura: "); Serial.print(t); Serial.println("°C");
  } else {
    Serial.println("Error al leer el sensor DHT22");
  }
  delay(2000);
}
